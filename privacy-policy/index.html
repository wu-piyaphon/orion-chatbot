Privacy Policy

Last updated: June 2025

1. Overview

This application integrates with the Meta Messaging API (Facebook Messenger and Instagram) for development and testing purposes. It operates as a webhook receiver and message sender on behalf of a connected Facebook Page and Instagram Professional Account.

2. Data We Collect

During operation, the application may process:

  • Sender IDs (PSID / IGSID) assigned by Meta's platform
  • Message text and attachment types sent to the connected Page or account

3. How We Use Data

Received messages are logged to the application console for debugging only. No personal data is written to a database, shared with third parties, or retained beyond the current server session.

4. Data Retention

No persistent storage is used. All processed data exists only in memory while the application is running and is discarded when the server stops.

5. Third-Party Services

This application communicates with the Meta Graph API (developers.facebook.com) solely to receive and send messages. Meta's own Privacy Policy governs data handled on their platform.
